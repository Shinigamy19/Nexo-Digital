# Código de Conducta — Nexo Digital

## Nuestro compromiso

Nexo Digital es una comunidad inclusiva. Nos comprometemos a hacer del
proyecto un entorno libre de acoso para todos, sin importar edad, tamaño
 corporativo, discapacidad, etnia, características sexuales, identidad
de género, apariencia, religión, u orientación sexual.

## Comportamientos esperados

- Uso de lenguaje respetuoso y considerado
- Respeto por las opiniones y experiencias de otros
- Aceptación de la crítica constructiva
- Enfoque en lo que es mejor para la comunidad
- Empatía hacia otros miembros de la comunidad

## Comportamientos no aceptados

- Lenguaje o imágenes sexuales y atención sexual no deseada
- Comentarios provocadores, insultantes o despectivos
- Acoso público o privado
- Publicación de información privada sin permiso
- Otras conductas que razonablemente podrían considerarse inapropiadas

## Alcance

Este código de conducta aplica a todos los espacios de la comunidad,
incluyendo:

- Repositorio de GitHub (issues, pull requests, discusiones)
- Grupo de WhatsApp de la comunidad
- Perfil público del sitio (`/u/username`)
- Cualquier otro espacio de comunicación del proyecto

## Aplicación

Los maintainers están obligados a aplicar este código de conducta en
todos los espacios del proyecto. Pueden eliminar, editar o rechazar
comentarios, commits, código, ediciones de wiki, issues y otros
contribuciones que no se alineen con este código, y pueden prohibir
temporal o permanentemente a cualquier usuario por comportamientos
que consideren inapropiados, amenazantes, ofensivos o dañinos.

## Aplicación

Los maintainers que no sigan o apliquen el código de conducta de
manera buena fe podrán enfrentar consecuencias temporales o
permanentes según la gravedad.

## Reporte

Para reportar comportamientos que violen este código, contactá a los
maintainers por el [grupo de WhatsApp](https://chat.whatsapp.com/Hbm9wubODmw2ycWx5DTcMZ)
o abrí un [GitHub Security Advisory](https://github.com/Shinigamy19/Nexo-Digital/security/advisories/new).

Todos los reportes serán revisados de forma confidencial.

## Atribución

Este código de conducta está basado en el [Contributor Covenant](https://www.contributor-covenant.org),
versión 2.1.
